One-TimePad(OTP) VPN Project (Group project)

It is the encryption of a VPN network using OTP (One-TimePad).

Assertion

Introduction

The goal of this project is to create an OTP encryption plugin for the LightRouter VPN system. The LightRouter system was a product already developed by our client, which is a light application with additional modules that can be added to increase its functionality. One-Time Pad (OTP) encryption technique uses a random key which has the same length as the message (1:1), so in theory, OTP protocol cannot be cracked. This helps ensure safe, secure transmission of data via VPN. Messages that are encrypted using OTP is considered very safe, ensuring safe working environment for users, and peace of mind for administrators.
